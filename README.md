This repository contains the UI framework implementation for the **Remainders** productivity/reminder application.

https://www.figma.com/design/cY6rLXiNzxbswFhRvtkVLg/Remainders-app-UI-framework

## Installation

Clone the repository:

```bash
git clone https://github.com/sathvikred1/ui-for-the-remainders-app.git
cd ui-for-the-remainders-app
```

Install dependencies:

```bash
npm install
```

---

##  Development

Start the development server:

```bash
npm run dev
```

The app will be available at:

```
http://localhost:3000
```

# Remainders

Remainders is a cross-platform productivity application focused on structured task tracking, smart scheduling, and seamless synchronization across devices.

This repository contains the UI foundation and scalable component architecture that powers the Remainders application interface.

## Overview

Remainders is designed with a clean, structured design system to support:

- Mobile (Android & iOS)
- Desktop (Windows)
- Responsive layouts
- Light and Dark themes
- Real-time sync-ready UI states

The UI framework emphasizes modular components, consistent spacing, and scalable architecture for future feature expansion.
